# Sample Job Description

## Role

Senior Platform Engineer

## Company

Company A (Example)

## Summary

We're looking for a senior engineer to build and operate shared infrastructure and internal developer tooling. You'll partner with product and security to improve reliability, deployment velocity, and observability.

## Responsibilities

- Design and operate cloud infrastructure and networking
- Build CI/CD pipelines and self-service developer workflows
- Improve observability with metrics, logs, traces, and incident response practices
- Collaborate with security on access controls and compliance needs
- Mentor engineers and lead small initiatives end-to-end

## Requirements

- Strong experience with AWS or equivalent cloud platforms and infrastructure as code
- Experience operating production systems, including on-call, incident management, and postmortems
- Familiarity with container orchestration and service networking
- Strong Python or Go skills for automation and tooling

## Nice to Have

- Experience with multi-account cloud governance
- Experience with OpenTelemetry or similar instrumentation
- Experience building internal platforms for developer productivity
